Italics and bold

Italics: _italic text_ or *more italic text*

Output: italic text or more italic text

Bold: __bold text__ or **more bold text**

Output: bold text or more bold text

Horizontal lines

A horizontal rule can be made by typing three or more dashes:

---------------

Headers

Headers of various sizes:

# level 1 header (H1) # or ### level 3 header (H3) ###

Output:

level 1 header (H1)

or

level 3 header (H3)

Links

[This is a link](http://google.com)

Output: This is a link

Block quotes

> This is quoted.
This is also quoted.

This is not quoted.

Output:

This is quoted. This is also quoted.

This is not quoted.

Escaping

To ensure that any one of these rules does not apply, simply place a backslash before the character. For example:

\*\*Text in stars, not in bold or italics\*\*

Output: **Text in stars, not in bold or italics.**
